Understanding iFrames

An iFrame (inline frame) is an HTML element that allows you to embed another HTML page within the current page.

Basic iFrame Example

iFrame Attributes

  • src: Specifies the URL of the page to embed
  • width: Sets the width of the iframe
  • height: Sets the height of the iframe
  • title: Provides accessibility information
  • frameborder: Specifies whether to display a border (0 or 1)
  • scrolling: Controls scrollbars (yes, no, auto)
